# 400 Free Resource Websites

A curated list of 400 genuinely useful free tools, organized by category. Every link below has been checked and points to the real, authentic site. Honest one-line reviews included — freemium tools and legally grey sites are flagged so you know exactly what you're getting.

---

## Media, Streaming & Audio

1. **[cobalt.tools](https://cobalt.tools/)** — Download video/audio from almost any social platform. Clean, no ads, no watermarks. Genuinely the best in its class.
2. **[justwatch.com](https://www.justwatch.com/)** — Tells you which streaming service has the movie/show you want. Saves you hopping between apps. Totally free.
3. **[archive.org](https://archive.org/)** — The Internet Archive: old web pages, plus free public-domain movies, music, books and software. A civilizational treasure.
4. **[archive.ph](https://archive.ph/)** — Permanently snapshot any web page. Handy for citations and dead-link insurance (also bypasses some paywalls — use responsibly).
