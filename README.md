# 200 Free Resource Websites That Should Be Illegal

A curated list of 200 genuinely useful free tools, organized by category. Every link has been checked and points to the real, authentic site. Honest one-line reviews included — freemium tools and legally grey sites are flagged so you know exactly what you're getting.

*v2 (2026-07): expanded from 100 → 200. Original 100 audited 2026-05-29; the 100 additions were verified 2026-07-04. Full audit details in [audit.md](audit.md).*

## Media, Streaming & Audio
